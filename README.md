# 📱 ADVANCED MOBILE PRICE PREDICTION SYSTEM

A premium, production-grade web application for predicting smartphone valuations in the Indian market using Machine Learning.

---

## 🚀 HOW TO RUN THE PROJECT (STEP-BY-STEP)

Follow these instructions to set up and run the application on your local machine:

### 1. PREREQUISITES
Ensure you have **Python 3.9+** installed on your system.

### 2. CLONE OR NAVIGATE TO THE DIRECTORY
Open your terminal and navigate to the project folder:
```bash
cd /Users/rajvishvakarma/ml_project/ss1/Mobile-Price-Prediction
```

### 3. CREATE A VIRTUAL ENVIRONMENT (RECOMMENDED)
It is best practice to use a virtual environment to avoid dependency conflicts:
```bash
python3 -m venv venv
```

### 4. ACTIVATE THE VIRTUAL ENVIRONMENT
- **On macOS/Linux**:
  ```bash
  source venv/bin/activate
  ```
- **On Windows**:
  ```bash
  .\venv\Scripts\activate
  ```

### 5. INSTALL DEPENDENCIES
Install the required Python packages using the provided `requirements.txt`:
```bash
pip install -r requirements.txt
```

### 6. RUN THE APPLICATION
Launch the Streamlit web interface:
```bash
streamlit run Advanced_Mobile_Price_App.py
```

### 7. ACCESS THE UI
Once the command is running, the app will be available at:
👉 **Local URL**: `http://localhost:8501`
