# 📱 SMARTPHONE PRICE INTELLIGENCE

A high-performance, machine-learning-driven platform for predicting smartphone market valuations in India. Featuring a premium "Glassmorphic" interface and real-time inference.

---

## 🚀 QUICK START (LOCAL RUN)

If you have the environment set up, simply run:
```bash
source venv/bin/activate
streamlit run Advanced_Mobile_Price_App.py
```

### 🛠️ FULL INSTALLATION
1. **ENVIRONMENT**: Create and activate a virtual environment.
2. **INSTALL**: `pip install -r requirements.txt`
3. **LAUNCH**: `streamlit run Advanced_Mobile_Price_App.py`

---

## ✨ KEY FEATURES
- **AI PREDICTION**: Uses a trained Random Forest Regressor to provide highly accurate price estimates.
- **INTERACTIVE INSIGHTS**: Visualize market trends and hardware correlations with Plotly.
- **DEVICE COMPARISON**: Compare two different smartphone configurations side-by-side.
- **PREMIUM UI**: Modern, centered, and uppercase design with dynamic Light/Dark mode compatibility.

## 📦 TECH STACK
- **Language**: Python 3.9+
- **Machine Learning**: Scikit-Learn, Pandas, NumPy
- **Frontend**: Streamlit, Custom CSS (Glassmorphism)
- **Visualization**: Plotly Express
